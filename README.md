# Smash Arena ID Scanner

スマブラ SP の**専用部屋の ID** を自動で読み取って、クリップボードへコピーしてくれる **Windows 専用**ツールです。  
OBS Studio で映している画面を常にチェックして、部屋 ID が画面に出た瞬間にコピー＆通知します。

## 📥 ダウンロード
最新版はこちらからどうぞ → https://github.com/manmenmishiikun/Smash-Arena-ID-Scanner/releases

---

## 必要なもの

- **PC の OS**: Windows 10 または 11（64 ビット推奨）
- **OBS Studio**: 配信・録画に使っているソフト。部屋待ち画面をキャプチャするために使います
  - バージョン **28 以降**なら追加プラグイン不要です（それより古い場合は別途 `obs-websocket` プラグインが必要になることがあります）

---

## 配布ファイルの使い方

1. ZIP ファイルを解凍します。
2. `SmashArenaIDScanner.exe` をダブルクリックして起動します。

> 初回起動時、exe と同じフォルダに `config.json` という設定ファイルが自動で作られます。次回からは前回の設定が引き継がれます。

> 初めて実行したとき、Windows の「SmartScreen」やウイルス対策ソフトが警告を出す場合があります。配布元を信頼できる場合のみ「詳細情報」→「実行」などで進めてください。

## 最初にやること：OBS の WebSocket 設定

このアプリは OBS の「WebSocket」という機能を使って画面の画像を取得しています。  
以下の手順で OBS 側の設定をオンにしておいてください（**1 回だけ**の作業です）。

1. OBS Studio を起動します。
2. 上のメニューから **「ツール」→「WebSocket サーバー設定」** を開きます。  
   （英語表示の場合は **Tools → WebSocket Server Settings**）
3. **「WebSocket サーバーを有効にする」** にチェックを入れます。
4. **ポート番号** を確認します。特に変えていなければ **4455** のはずです。変えている場合はその番号をメモしておきます。
5. **「認証を有効にする」** にチェックしている場合は、設定したパスワードもメモします。チェックしていない場合はパスワードは空のままで大丈夫です。
6. 「OK」で閉じます。

## アプリの使い方

### ステップ 1：OBS に接続する

アプリを起動すると、最初は**「OBS WebSocket 接続設定」カード**が開いた状態になっています。

| 入力欄 | 何を入れるか |
|--------|-------------|
| **IP アドレス** | OBS と同じ PC で使うなら **`localhost`** のまま。別の PC の OBS に繋ぐ場合はその PC の IP アドレス |
| **ポート番号** | OBS の WebSocket 設定で確認した番号（多くの場合は `4455`） |
| **パスワード** | OBS で「認証を有効にする」にした場合のみ入力。していない場合は空のままで OK |

入力したら **「OBS に接続」** ボタンを押します。

接続が成功すると、画面が切り替わり **「対象ソース」カード** が開いた状態になります（接続設定は自動で畳まれます）。

### ステップ 2：部屋待ち画面のソースを選ぶ

「■ 対象ソース」の一覧に、OBS のソース（キャプチャ中の画面）が表示されます。  
**部屋待ち画面が映っているソース**を選んでください。

### ステップ 3：監視を開始する

**「監視を開始」** ボタンを押します。  
OBS の画面をチェックし続け、部屋 ID が見つかった瞬間に**クリップボードへ自動コピー**＋**通知音**で教えてくれます。

> 部屋 ID が確定したら Win+V（クリップボード履歴）にある古い部屋 ID の記録も自動で削除されます（環境によっては削除されないこともあります）。

監視を止めるときは **「監視を停止」** を押してください。  
接続設定を変えたいときは **「再接続」** ボタンで最初の接続画面に戻れます。

## 画面の見方

### 中央の大きな ID 表示

直近で読み取った部屋 ID が大きく表示されます。**クリックするといつでも再コピー**できます。

### 監視ボタンの横にある小さな丸ランプ

| 色 | 意味 |
|----|------|
| 灰色 | 監視していない（停止中） |
| 赤 | 監視中 |
| 白（一瞬） | 部屋 ID をコピーした瞬間 |

### ROOM / ID ランプ

画面右上あたりにある 2 つの小さなランプです。

| ランプ | 意味 |
|--------|------|
| **ROOM** | 専用部屋の画面を検知中 |
| **ID** | 部屋 ID（5 桁）を読み取り済み |

どちらも点灯していれば正常に検知できています。

### 下部のトグル・チェックボックス

| 項目 | 内容 |
|------|------|
| **🔔 通知音** | 部屋 ID を見つけたときの音のオン / オフ |
| **📌 常に最前面** | アプリウィンドウを常に他のウィンドウの手前に表示 |
| **⚡ 自動接続** | 次回起動時に自動で OBS に接続して監視まで開始する |

### 🕒 履歴ボタン

今回の起動中に読み取った過去の部屋 ID の一覧から選んでコピーできます。

## うまくつながらない・動かないとき

- OBS が起動しているか確認する
- OBS の WebSocket が**有効になっているか**確認する（「ツール → WebSocket サーバー設定」）
- **ポート番号**が OBS とアプリで同じか確認する
- **パスワード**を正しく入力しているか確認する（認証オフなら空欄）
- **Windows のファイアウォール**やセキュリティソフトが通信をブロックしていないか確認する
- 別の PC の OBS に繋ぐ場合は `localhost` ではなく**その PC の IP アドレス**を入力する
