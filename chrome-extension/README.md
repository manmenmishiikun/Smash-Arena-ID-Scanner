# Smash Arena ID Bridge（Chrome 拡張）

**Smash Arena ID Scanner**（デスクトップ）が `127.0.0.1` で配信する **Server-Sent Events（SSE）** に接続し、受け取った **部屋コード** を `smashmate.net` の **rate ページ**上の入力欄へ反映する補助拡張です。

## 重要な前提（混同禁止）

- **SSE の `data` 行**は、ゲーム画面 OCR 由来の **部屋コード**（デスクトップが確定した文字列）です。**この文字列をそのまま**入力欄に渡します。拡張側では **文字種の正規表現やルールを再実装しません**（必要なら **長さ上限のみ**）。
- **URL パスに含まれる数字**（例: `https://smashmate.net/rate/1136115/` の `1136115`）は **プロフィール／レート側の識別子**であり、**部屋コードではありません**。桁数も固定ではありません。実装・本 README では **URL の数字を部屋 ID の定義に使いません**。

## 自動化の範囲（やること／やらないこと）

| 項目 | 内容 |
|------|------|
| 行うこと | 対象 URL のページで、部屋コード入力欄の **`value` への反映**、軽い **アウトライン表示**（数百 ms）。**拡張の設定で ON にした場合のみ**、続けて **`matching_post_btn`**（表示は「設定」「変更」など）を **`click()`** します。クリックは安全策として **最短15秒のクールダウン**と**同一IDの短時間再クリック抑制（2分）**を適用します。 |
| 行わないこと | **サイトの確認ダイアログ等の操作**、**フォームの `submit()` をコードから直接叩くこと**。**ボタンの自動クリック**は **既定 OFF**（利用者が拡張のチェックボックスで明示的に有効化したときだけ実行）。 |
| 重複 | デスクトップから **同一部屋コードが繰り返し**届く場合があります。コンテンツスクリプトは **直前と同じ文字列なら入力をスキップ**します（不要な上書き・視覚フィードバックの連発を抑止） |

## 対象ページ（それ以外では DOM を触りません）

- **コンテンツスクリプト**は `https://smashmate.net/rate/*` にだけ注入します（トップや別セクションではスクリプト自体を読み込みません）。**部屋コードの自動入力**はさらに `location.pathname` が **`/rate/<数値>/` 形式**（`shared-path.js` の `isRateProfilePath` と同じ）のときのみです。
- **SSE 接続**は「ローカル SSE に接続する」が ON かつ、**上記 rate プロフィール URL のタブが 1 つ以上あるときだけ**バックグラウンドが張ります（rate ページをバックグラウンドのまま別サイトを見ている間は接続を維持します。rate 用タブをすべて閉じる／別 URL に遷移すると切断します）。対象外のページでは **localhost へも繋がず**、DOM も触りません。

実サイトの URL が末尾スラッシュ付き／なしのどちらでも、上記パターンにマッチすれば動作します。

## ローカルブリッジとの合わせ方

- **URL**: `http://127.0.0.1:<ポート>/events`（パスは **`/events`** 固定）
- **ポート**: デスクトップの **「連携ポート」** と **拡張のオプションのポート**を **同じ値**にしてください。既定は **2206** ですが、**ハードコードせず**オプションで変更します。
- **デスクトップ側**: 「拡張機能と連携」が ON かつ **OCR 監視中**のときだけ SSE が Listen されます。監視を止めるとストリームが切れます。

## SSE の読み方（Service Worker）

- **`EventSource`** は環境によって挙動差が出ることがあるため、本拡張では **`fetch` + `ReadableStream`（`getReader`）** で本文を読み、**`data: ...` と空行区切り**をパースしています。切断検知・再接続間隔の制御もしやすいです。

## 切断・再接続

- デスクトップで監視停止・連携 OFF・アプリ終了などで **SSE が切れます**。
- 拡張は **指数バックオフ**（初期 1 秒、最大 30 秒）で **再接続**を試みます（接続 ON かつ **rate プロフィールタブが少なくとも 1 つある間**）。
- ツールバーアイコンの **バッジ**: 接続できているとき `OK`（緑系）、エラー時 `!`（赤系）、接続 OFF 時は非表示です。

## 権限・プライバシー

- **`http://127.0.0.1/*`**: ローカル SSE への接続のみ。
- **`https://smashmate.net/rate/*`**: 対象タブへメッセージを送り、コンテンツスクリプトで入力を更新するため（スコープ最小化）。
- **収集・外部送信**: 部屋コードを **当拡張のサーバーに送る処理はありません**。スマメイトの利用規約・ポリシー・サーバー負荷は **利用者の責任**です。本拡張は **一般目的のローカル連携クライアント**として提供します。

Chrome Web Store に公開する場合は、上記の権限理由とデータ非送信を **ストアの説明文**や **プライバシーポリシー**に書くと審査に通りやすくなります。

## 配布・動作環境

- **拡張単体**: ネイティブコードなし。**Chrome / Edge 等 Chromium（Manifest V3）** が動く環境で、**CPU/GPU を特別に要求しません**。
- **配布**: Chrome Web Store、または ZIP を **開発者モードで「パッケージ化されていない拡張機能を読み込む」** など。
- **OCR デスクトップアプリ**は別リポジトリで **Windows 前提**のため、**部屋コード生成まで含めた一式**は OS が分かれます。拡張のみならブラウザさえあれば利用できます。

## インストール（開発）

1. Chrome で `chrome://extensions` を開く。
2. 「デベロッパーモード」を ON。
3. 「パッケージ化されていない拡張機能を読み込む」で、この `chrome-extension` フォルダを指定。
4. **ツールバーの拡張アイコンをクリック**すると設定パネル（ポップアップ）が開きます。ここで **ポート**と **ローカル SSE に接続**を設定します（従来どおり `chrome://extensions` の「拡張機能のオプション」からも同じ画面を開けます）。アイコンがパズルメニュー内だけのときは、パズルアイコンから本拡張を **ピン留め**すると常に表示されます。

## 手動確認の目安

- デスクトップ: 「拡張機能と連携」ON、**連携ポート**例: 2206、**監視中**。
- 拡張: 同じポート、**ローカル SSE に接続** ON。
- `https://smashmate.net/rate/<数値>/` を開き、OCR が部屋コードを確定すると入力欄に反映されること。
- 監視停止後、バッジがエラー側に寄る／再接続を繰り返すこと。
- **対象外 URL** では入力が変わらないこと。

## DOM セレクタ（要メンテ）

送信ボタン（`matching_post_btn`）や入力欄の **HTML はサイト改修で変わりうる**ため、定数は [`SELECTORS.md`](SELECTORS.md) および [`content.js`](content.js) 先頭を参照してください。ボタン文言は **「設定」／「変更」などに依存せず**、クラスのみで判定します。部屋ID欄は実サイトで **`type="url"`** のため、入力は **`input.room_matching_id[name="room_matching_id"]`** を最優先しています。
